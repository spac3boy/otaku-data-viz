# Otaku Data Viz homepage package

This package contains the current `index.html` homepage for the GitHub repo:

https://github.com/spac3boy/otaku-data-viz

## Included structure

```text
otaku-data-viz/
  index.html
  .nojekyll
  README_DEPLOY.md
  dragonball-character-sociogram/
    .gitkeep
```

The homepage Dragon Ball project card points to:

```text
/otaku-data-viz/dragonball-character-sociogram/
```

For that link to work on GitHub Pages, your repo should eventually contain:

```text
dragonball-character-sociogram/
  index.html
```

## GitHub Pages expectation

If this is a project page repo named `otaku-data-viz`, GitHub Pages will usually publish it at:

```text
https://spac3boy.github.io/otaku-data-viz/
```

The Dragon Ball page URL will be:

```text
https://spac3boy.github.io/otaku-data-viz/dragonball-character-sociogram/
```
