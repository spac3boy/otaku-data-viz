Why map fictional worlds?
Fan culture has timelines, systems, rivalries, eras, and invisible architectures worth mapping.
ファン・カルチャー・マッピング

Otaku Data Viz is a collection of fan-made data visualizations by Michael Buquet. Each project uses public research, structured data, and interactive design to make complex fictional worlds easier to explore.

Every project can have its own interface and personality. The main site acts like an issue cover, archive, and table of contents for the data stories.

This site is unofficial and not affiliated with any anime, manga, game, publisher, studio, or rights holder.
