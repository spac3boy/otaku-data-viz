ファン・カルチャー・マッピング

Otaku Data Viz is a collection of fan-made data visualization experiments by Michael Buquet. Each project uses public research, structured data, and interactive design to make complex fictional worlds easier to explore.

Every data-viz project can have its own interface and personality. The main site stays separate: a retro-modern index, archive, and launchpad for the experiments.

This site is unofficial and not affiliated with any anime, manga, game, publisher, studio, or rights holder.
